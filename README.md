# SDNMap
SDNMap is an OpenFlow graphing tool built on wxPython. Run SDNMap on a floodlight host to graph network topology.

### Dependencies
graphviz, python-wgtk2.6, networkx, numpy, matplotlib, pygraphviz

### Usage
Run on a floodlight host

    python main.py
